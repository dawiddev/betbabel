# Betbabel

This is a simple web app, it is meant to help users convert their betslips from one bookie to another using booking codes.

### Tech Stack

-HTML
-CSS
-NodeJS

## How to run? 

1. Clone repo  

2. run npm install 

3. run any of the scripts in *package.json* depending on what you are doing.
